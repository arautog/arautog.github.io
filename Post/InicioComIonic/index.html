Iniciando com Ionic

Iniciando com Ionic

Olá galera, nesse post irei falar sobre Ionic, que é um framework front-end para mobile, quem já teve experiencia com angularJS, não terá dificuldades com com esse framework, ele utiliza basicamente HTML, CSS, JAVA-SCRIPT, AngularJS que é essencial para criar seus componentes e Cordova como integração com os recursos nativos dos dispositivos moveis, que rodam em sistemas operacionais iOS7+ e Android 4.4+.

Vantagens de usar o Ionic

  • Baixa curva de aprendizado
  • Custo de desenvolvimento menor
  • Compatibilidade entre dispositivos
  • Arquitetura MVC
  • Vários componentes já prontos similares aos nativos
  • Extremamente produtivo
  • Muito material na internet

Antes de começar a usar !!!

De uma olhadinha de como funciona o framework (AngularJS), isso facilitará o seu desempenho na hora que estiver criando seu app em Ionic pois como disse a pouco AngularJS é a base do ionic.

É necessário baixar e instalar o nodeJS (aqui)
Para verificar já se possui o node instalado em sua maquina execute o comando no seu shell ou prompt de comando :

> node -v

como retorno terá a versão do node instalada em sua maquina, isso se ele estiver instalado,
também verifique qual versão do npm, para ver se está tudo correto, com o comando:

> npm -v

Quanto instalei pela primeira vez em minha maquina tive alguns problemas com nodeJS, que foi resolvido abrindo o prompt de comando em modo administrador. encontrei um link interessante onde você faz algumas configurações e pode usar o nodeJS sem ser administrador.(Link)

Nesse momento também é interessante você já instalar o Gulp, que iremos utilizar um pouco mais pra frente para compilar o SCSS, isso mesmo, o ionic utiliza o preprocessador Sass para gerar o CSS.

> npm install -g gulp

Ambiente de desenvolvimento

Você pode utilizar qualquer editor de texto, ou qualquer ide que te ajude no desenvolvimento de HTML, CSS, Java-Script, como exemplo posso citar o Sublime, que é gratuito e conta com uma gama de plugins, eu particularmente utilizo WebStorm, é pago mais já estou acostumado com ele.
O ionic já conta com emuladores para simular seu projeto, porem se você estiver desenvolvendo para IOS é necessário estar utilizando um sistema operacional OSX, é interessante também já ter instalado no computador as sdks dos mesmos.

Instalando o Ionic

Para instalar o Cordova e o Ionic, digite no shell ou no prompt de comando :

> npm install -g cordova ionic

Apos a instalação do cordova e ionic ja podemos criar um projeto, existe 3 templates que podemos utilizar, blank, tabs e sidemenu

Para criar um projeto com o template blank execute o comando:

> ionic start nomeMeuAplicativo blank

Já para criar modelo de tabs, execute o comando: 

> ionic start nomeMeuAplicativo tabs

E por final para criar template de sidemenu execute o comando: 

> ionic start nomeMeuAplicativo sidemenu

Substitua “nomeMeuAplicativo” pelo nome que quiser.
Abaixo você pode verificar os 3 templates de como serão na forma inicial.

Agora para configurar o Sass do Ionic execute o comando:

> ionic setup sass

No arquivo app,js é que utilizamos para configurar as rotas do AngularJS, junto com ele ficam nossos services.js e controllers.js que são utilizados para melhorar a estrutura aplicando a logica da arquitetura MVC.
Na raiz do projeto temos o arquivo config.xml utilizado para configuração de build, arquivo bower.json onde configura as dependências que o bower irá gerenciar, o arquivo gulpfile temos algumas tarefas do Ionic e no arquivo package.json as dependencias do nodeJS.

Startando o app para testes

Depois de ter feito as configurações acima, basta ir no shell ou prompt de comando e executar a linha:

> ionic serve

Pronto, agora você ja pode ir no seu navegador e ver a aplicação rodando, no endereço “localhost:8100”

Concluindo
